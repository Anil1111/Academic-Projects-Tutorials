## Sample
Uncomment the js files in the script tag of `./index.html`

1. `npm start` at ./ to run the project
2. `tsc -w` at ./ to compile the `src/*.ts` files into `dist/*.js`
3. `nodemon file.js` from `dist/*.js` file
